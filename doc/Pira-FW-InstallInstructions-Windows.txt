Pira-smart-firmware
Mbed-CLI install instructions on Windows

1. Install Python 2.7 and add it to windows system path
2. Install Git and Mercurial
3. Install GNU ARM embedded toolchain version 6 (or 7)
2. > pip install mbed-cli
3. > mbed import https://github.com/IRNAS/pira-smart-firmware.git
4. > cd pira-smart-firmware
5. > mbed config -G GCC_ARM_PATH "<gcc_compiler_bin_folder_path>"
6. > mbed deploy
7. > mbed compile -m RBLAB_BLENANO -t GCC_ARM
